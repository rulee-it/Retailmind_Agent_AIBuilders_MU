How the brain wires together.

Three layers. Five LLM calls. Strict tool isolation enforced in code.

i.

Router

An LLM classifies every query into one of six routes — INVENTORY · PRICING · REVIEWS · CATALOG · MULTI · GENERAL — and emits a structured decision. No keywords. No regex.

  • gpt-4o-mini · temp 0
  • structured output
ii.

Specialists

Four scoped agents. Each owns its tools and nothing else. The Inventory agent talks days-to-stockout. Pricing talks margin and percentile. Reviews summarises themes. Catalog returns formatted lists.

  • 2 tools each · 8 total
  • isolation by import
iii.

Supervisor

For cross-domain questions and the Daily Briefing, a fifth LLM call orchestrates 2+ specialists in parallel and stitches their outputs into one ranked recommendation — with citations.

  • gpt-4o · no tools
  • ranks by impact
